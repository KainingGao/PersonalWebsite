## Project Overview

At Iowa State University's NextGen PPT Lab, I developed an automated 3D hand model analysis system that replaced manual measurement workflows with a Python-based data processing pipeline. The system eliminated manual measurement errors and increased measurement speed by 400%.

### Key Technologies

- **Data Processing**: Python
- **Backend**: FastAPI
- **Cloud**: AWS-hosted services
- **Database**: PostgreSQL
- **AI Processing**: Integrated model-driven analysis workflows

### Responsibilities

- Built the automated measurement pipeline for 3D hand model analysis
- Maintained an AWS-hosted FastAPI backend for a healthcare research application
- Managed PostgreSQL data workflows used by the analysis system
- Collaborated with the research team to standardize protocols and improve data accuracy

## Impact

The project turned a slow, manual measurement process into a faster and more consistent system for research use. It also gave me deeper experience connecting data pipelines, backend services, cloud operations, and research requirements in one production-grade application.
